About This Page...

The purpose of this page is to demonstrate the basic functionality of creating a dynamic table and updating the data via API calls.
  • "Add Stock" tab: the table leverages an API offered by iex cloud. We can add stocks via the "Add to Portfolio" search bar.
  • "Delete Stock" tab: We can delete any stocks that we no longer want.
  • "Get Stock Quote" search bar: Entering a ticker returns basic stock information such as price, market cap, YTD percentage change, etc.

Next steps:
